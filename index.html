3D Displacement Sensor Fusion

+ Modeled 3D displacement using data from a MEMS Sensor (Accelerometer, Gyroscope, Magnetometer) created by STMicroelectronics called SensorTile together with the Nucleo development board.

What I achieved

Left image: Greatly reduce the sensors' noise as shown in the Noise Spectrum graph.
Right image: Improved the accuracy of 3D Displacement Calculation using only IMU sensors. Minimized the error existed in the red trajectory to a more accurate calculation as indicated by the black trajectory.
